# EAD_group_assignment

Spring-Boot + MySQL + React js application and GIT for version controlling.

Client folder contains  the frontend source codes and the server folder contains the backend source code related to the project.
There are some screenshots of the web pages contain screenshot folder.
This ER includes a straightforward ER diagram created to help understand the application's structure.

Password is not set (default) in SQL
Username is set as 'root' (default)

If username and password is not set properly application will fail to run.
